# LocalVRAM Logo Options

This folder contains three logo directions for the `LocalVRAM` brand:

- `tech-minimal/`
- `terminal-hacker/`
- `hardware-lab/`

Each option has:

- `logo.svg` (wordmark)
- `logo-mark.svg` (icon only)

Shared asset:

- `shared/ollama-verified-pill.svg` (homepage label, not part of the logo)

## Usage notes

- Keep logo text exactly `LocalVRAM`.
- Do not attach any subtitle to the logo itself.
- Place `Ollama Verified` as a separate homepage badge near hero/nav.

Additional hardware-lab style variants:

- `logo-options/hardware-lab-v1-chip-core/`
- `logo-options/hardware-lab-v2-cooling-ring/`
- `logo-options/hardware-lab-v3-rack-node/`

Each includes `logo.svg`, `logo-mark.svg`, and `favicon.svg`.
